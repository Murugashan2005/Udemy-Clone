# Udemy Clone

A simple Udemy Clone website built using **HTML** and **CSS**. This project recreates the basic layout and design of the Udemy homepage with a clean and responsive user interface.

## Features
- Responsive homepage design
- Navigation bar with search box
- Course categories section
- Featured courses display
- Clean and modern UI
- Beginner-friendly HTML and CSS structure

## Technologies Used
- HTML5
- CSS3

## Purpose
This project was created to improve my front-end development skills by practicing HTML and CSS, including layouts, styling, and responsive web design.
